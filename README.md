# Typing Game

A modern typing speed test application inspired by Monkeytype, built with **React**, **TypeScript**, and **Go**.

The project is being developed using professional software engineering practices, including project planning, documentation, Test-Driven Development (TDD), and GitHub workflows.

---

## Project Goals

The primary objective is to create a fast, responsive, and enjoyable typing experience while demonstrating modern frontend architecture and software development practices.

The application will allow users to:

* Practice typing using randomly generated words
* Measure typing speed (WPM)
* Measure typing accuracy
* View typing statistics
* Improve typing performance over time
